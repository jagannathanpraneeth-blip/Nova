# ⚡ NOVA MULTITASKING GUIDE

## 🚀 PERFORM MULTIPLE TASKS AT ONCE!

Nova can now understand and execute multiple commands in a single sentence. This allows for true multitasking and automation.

### 🔗 HOW TO USE

Use keywords like **"and"**, **"then"**, **"also"**, **"after that"** to chain commands.

#### Example 1: Work Setup
```
"Nova, open chrome and open notepad and play lofi music"
```
**Result:**
1. Opens Chrome
2. Opens Notepad
3. Plays music on YouTube
**ALL AT THE SAME TIME!**

#### Example 2: Research & Note Taking
```
"Nova, analyze screen and create note with summary"
```
**Result:**
1. Analyzes your screen content
2. Creates a note with the summary

#### Example 3: Quick Actions
```
"Nova, minimize all and take screenshot"
```
**Result:**
1. Minimizes all windows
2. Takes a screenshot of your desktop

---

## ⚙️ HOW IT WORKS

1. **Command Parsing:** Nova intelligently splits your sentence into individual tasks.
2. **Task Queue:** Each task is added to a parallel execution queue.
3. **Worker Threads:** Nova has 3 background workers that execute tasks simultaneously.
4. **Non-Blocking:** You can keep talking to Nova while tasks are running!

---

## 💡 TIPS FOR BEST RESULTS

- **Keep it simple:** "Do X and Do Y" works best.
- **Wait for completion:** If tasks depend on each other (e.g., "open app AND type"), Nova tries to run them in order, but for strict timing, separate commands might be safer.
- **Mix and Match:** You can mix any commands:
  - "Open WhatsApp and send message"
  - "Play music and open calculator"
  - "Analyze screen and read it"
