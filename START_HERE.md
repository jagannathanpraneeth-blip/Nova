## 🛠️ IF SOMETHING DOESN'T WORK

### Voice too fast/slow?
Edit `data/config.json`:
```json
{"speech_rate": -1}  // -10 (slowest) to 10 (fastest)
```

### Typing not working?
- Make sure you click in target app within 2 seconds
- Or use "start dictation" for continuous typing

### YouTube not clicking?
- Wait full 7 seconds for page to load
- If screen is unusual size, it may miss - just click manually

### Screen vision says "no API key"?
- Run `setup_api_key.bat`
- Or set manually: `setx OPENAI_API_KEY "sk-..."`
- Restart terminal after setting

### Commands not recognized?
- Speak clearly
- Check logs in GUI window
- Try rephrasing command
