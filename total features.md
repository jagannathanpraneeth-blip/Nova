# Nova - Total Feature List

## 🔥 The "Fire" Update (Advanced Capabilities)
1.  **Intelligent Safety Layer**: Prevents accidental execution of dangerous commands (e.g., "uninstall python", "delete file") by intercepting them and requiring verbal confirmation.
2.  **Autonomous Installer Agent**: Automatically opens VS Code, searches for specific extensions (like "web dev" tools), installs them, and reloads the window.
3.  **Self-Healing Automation**: Detects when an automation task fails (e.g., installer hangs), closes the unresponsive window, and retries the operation automatically.
4.  **Context-Aware File Management**: Understands commands like "delete the last file I downloaded" by analyzing file timestamps in the Downloads folder.
5.  **Sequential Long-Task Automation**: Executes multi-step complex workflows, such as "set up a new project folder" (Create folder -> Open VS Code -> Create starter files -> Init Git -> Open Terminal).
6.  **Full Browser Automation**: Can perform end-to-end web tasks like "download react docs" (Search -> Navigate -> Save page as PDF).
7.  **Plugin System Architecture**: Modular design allowing for infinite expansion via specialized agent plugins.
8.  **Session Memory**: Remembers the last executed command, enabling natural follow-ups like "Do that again" or "Repeat that".
9.  **Multi-Agent Mode**: Orchestrates specialized agents (Code, Browser, System, Files) under a central "AI Brain".
10. **Enhanced Visual Feedback**: Displays real-time logs of internal thought processes and actions (e.g., "Delegating to InstallerPlugin...", "Waiting for confirmation").

## 🧠 Core Intelligence
-   **Universal AI Brain**: Central orchestrator that parses natural language and routes commands to the appropriate handler or agent.
-   **Command Chaining**: Supports multiple commands in one sentence (e.g., "Open Chrome and play music").
-   **Natural Language Understanding (NLU)**: Extracts intents and entities from spoken commands using fuzzy matching and keyword detection.
-   **Wake Word Detection**: Listens for "Nova" to activate.

## 🗣️ Voice & Interaction
-   **Speech-to-Text (STT)**: Converts voice input to text.
-   **Text-to-Speech (TTS)**: Responds verbally to the user with status updates and confirmations.
-   **Dictation Mode**:
    -   Continuous voice typing into any application.
    -   Supports punctuation commands ("comma", "period", "new line").
    -   Editing commands ("delete", "backspace", "scratch that").
    -   Formatting commands ("caps on", "all caps").
    -   One-shot typing: "Voice type [text] and send".

## 💻 System Control
-   **Application Management**:
    -   Open any application by name ("Open Notepad", "Launch Chrome").
    -   Close/Kill applications ("Close Calculator", "Kill Chrome").
    -   Smart Launch: Tries to find and open apps even if not explicitly mapped.
-   **Window Management**:
    -   Minimize all windows ("Show desktop").
    -   Maximize current window.
    -   Close current window ("Alt+F4").
    -   Switch windows ("Alt+Tab").
-   **System Power**:
    -   Shutdown, Restart, Sleep, Lock Workstation.
-   **Volume & Media**:
    -   Mute/Unmute system volume.

## 🖱️ Mouse & Keyboard Automation
-   **Mouse Control**:
    -   Move mouse (to coordinates or center).
    -   Click (Left, Right, Double).
    -   Scroll (Up, Down).
    -   Get mouse position.
-   **Keyboard Control**:
    -   Press specific keys ("Press Enter", "Press Esc").
    -   Execute shortcuts ("Ctrl+C", "Ctrl+V", "Ctrl+Z", "Win+D", etc.).
    -   Type text ("Type Hello World").

## 👁️ Screen Vision
-   **Screen Analysis**: Uses AI to analyze the current screen content ("Analyze screen", "What do you see?").
-   **Screen Reading**: Extracts and reads text from the screen ("Read screen").
-   **Visual Search**: Locates specific UI elements ("Find the submit button on screen").
-   **Screen Description**: Provides a detailed description of the visual interface.

## ⚡ Productivity Workflows
-   **WhatsApp**: Send messages via WhatsApp Web ("Send WhatsApp to Mom saying I'll be late").
-   **Email**: Open Gmail compose with pre-filled details ("Email John subject Meeting body Let's meet at 5").
-   **YouTube**: Search and play videos automatically ("Play lofi hip hop on YouTube").
-   **Google Search**: Perform web searches ("Search Google for Python tutorials").
-   **Wikipedia**: Search and read summaries ("Search Wikipedia for AI").
-   **Reminders**: Set system reminders ("Remind me to call back in 10 minutes").
-   **Note Taking**: Quickly create notes in Notepad ("Create a note saying buy milk").
-   **Screenshots**: Capture and save screenshots ("Take a screenshot named evidence").
-   **Math/Calculator**: Solve math problems verbally ("What is 25 times 4?").
