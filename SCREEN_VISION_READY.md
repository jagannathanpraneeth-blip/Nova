## 📝 TECHNICAL DETAILS

### How It Works:
1. You say "Nova, analyze screen"
2. Nova captures screenshot using `pyautogui`
3. Converts image to base64
4. Sends to OpenAI GPT-4 Vision API
5. Receives AI analysis
6. Speaks the description to you

### Privacy & Security:
- ✅ **Only when you ask** - Never captures automatically
- ✅ **Not saved** - Images sent directly to API, not stored
- ✅ **Secure** - API key stored in environment variable
- ✅ **Optional** - Works without API (just saves screenshot)
