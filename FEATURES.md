# Nova Features

## 🧠 Intelligence
- **Universal AI Brain**: Understands natural language and chains commands.
- **Session Memory**: Remembers previous commands ("Do that again").
- **Safety Layer**: Asks for confirmation before dangerous actions.

## 🔌 Plugins & Agents
- **Installer Agent**: Automates VS Code extension installation.
- **Project Agent**: Sets up new project folders with Git and starter files.
- **Browser Agent**: Automates web tasks like downloading documentation.
- **File Agent**: Manages files intelligently (e.g., "delete last download").

## 💻 Automation
- **Self-Healing**: Retries failed actions and recovers state.
- **Screen Analysis**: "What's on my screen?"
- **App Control**: Open/Close/Manage any application.
- **Mouse & Keyboard**: Full control via voice.

## 🗣️ Interaction
- **Voice Input/Output**: Hands-free operation.
- **Visual Feedback**: Real-time logs and status updates.
- **Dictation Mode**: Type with your voice.

## 🛠️ System
- **Plugin Architecture**: Easily add new features.
- **Configurable**: JSON-based configuration.
