# 💻 Vibe Coding with Nova

Nova now includes **Vibe Coding** capabilities, allowing you to generate full applications just by describing them.

## 🚀 How it Works
1.  **You Request**: "Nova, build a simple to-do list app with vanilla JS."
2.  **Nova Thinks**: The LLM architectures the app (HTML, CSS, JS structures).
3.  **Nova Builds**: 
    -   Creates a project folder on your Desktop.
    -   Writes all the code files.
    -   Creates a `setup_and_run.bat` script to install dependencies/run it.
4.  **Nova Launches**: Opens the project in VS Code and runs the setup script.

## 🗣️ Example Commands
-   *"Nova, create a python flask app that shows the current weather."*
-   *"Nova, build a personal portfolio website with a dark theme."*
-   *"Nova, make a simple game using HTML canvas."*

## 🎓 Study Assistant Mode
Since Nova can see your screen (`analyze_screen`), you can use it for help with college work:
1.  Open your VS Code or assignment.
2.  Say: *"Nova, look at this code and tell me why it's not working."*
3.  Say: *"Nova, explain this error message to me."*

Nova combines **Vision** + **Coding Intelligence** to be your pair programmer.
